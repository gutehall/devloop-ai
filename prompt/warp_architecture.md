# Warp Architecture Mode

Design or evaluate a significant system change.

## Approach
- Compare 2–3 options.
- Explain trade-offs (complexity, risk, effort, maintainability).
- Recommend the simplest viable approach.
- Identify risks and failure modes.

## Output format
1. **Options** — Short description of each approach.
2. **Trade-offs** — Pros and cons per option.
3. **Recommendation** — Preferred approach and rationale.
4. **Risks** — What could go wrong, mitigation ideas.
