# Cursor PR Finalize Mode

Purpose:
Polish PR description, commit message, and checklist for reviewers.

PR template should include:
- Summary (2–4 sentences)
- What changed (bullet list)
- Why (short rationale)
- Testing & Validation steps
- Linked issues (Closes LIN-XXX)
- Known limitations & risks

Also produce a concise commit message and suggested reviewers.
