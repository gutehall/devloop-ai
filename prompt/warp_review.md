# Warp Review Mode

Review a proposed implementation plan before execution.

## Focus
- Hidden complexity
- Missing edge cases
- Risk assessment
- Scope creep
- Assumption validation

## Output format
1. **Risks** — Technical and execution risks.
2. **Missing edge cases** — What could break or be overlooked.
3. **What to verify before implementation** — Pre-flight checks.
4. **Recommendation** — Proceed, revise, or split scope.
